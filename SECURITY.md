# Security Policy

## Supported Versions

Currently, we support the latest version of the main branch.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to **contato@scaledata.com.br**. All security vulnerabilities will be promptly addressed.

### Process

1.  **Report**: Send a detailed description of the vulnerability to the email above. Please include steps to reproduce if possible.
2.  **Acknowledge**: We will acknowledge receipt of your report within 48 hours.
3.  **Investigate**: We will investigate the issue and determine its impact.
4.  **Fix**: If confirmed, we will work on a fix and release it as soon as possible.
5.  **Disclose**: Once the fix is deployed, we may publicly disclose the vulnerability if appropriate.

Please do not publicly disclose the vulnerability until we have had a chance to fix it.
